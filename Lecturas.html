Lecturas

DISEÑO DE SITIOS WEB.

(Primera lectura)

El diseño de sitios web es una actividad que a través de los años se ha convertido en una herramienta informática muy elemental en la presentación de una página web, su construcción abarca el manejo de softwares que controla cierto tipo de versiones que permiten a un usuario acceder e interactuar de miles maneras en internet, cuando éstos soliciten acceder a algún tipo de información determinada Para este caso es importante conocer algunos sistemas controladores y versiones que van de la mano en la creación de páginas web y permite la edición colaborativa en usuarios que hayan construido un supositorio. Conozcamos a:

  • GIT-GITHUB
  • HTML5
  • CSS3

¿QUÉ ES GIT?

(segunda lectura).

Creado por Linux Torvalds, es un software confiable, versátil de fácil manejo en la utilización de elaboración de supositorios. Conocido también como controlador de versiones permite realizar cambios de códigos, fuentes y archivos en un proyecto determinado de una página web. Git para operar solo necesita archivos y recursos locales para funcionar, Por lo general no se necesita información de ningún otro computador de una red, debido a que sus operaciones aparece de forma inmediata, ya que su cambios quedan almacenados en el disco local. GIT como no necesita estar conectado a un servidor puede mostrar la búsqueda de archivos creado en una fecha determinada y mostrarla haciendo cálculos diferenciales localmente.

¿QUÉ ES GITHUB?

(tercera lectura).

GitHub es conocida como una plataforma de desarrollo colaborativa que permite la edición de proyectos en correlación a sistemas controladores de versiones como GIT. Se utiliza principalmente para la creación de código fuente de programas de computadora. Estos proyectos son almacenados de manera pública y privada según la necesidad del usuario.

¿Qué proporciona GITHUB?

GITHUB como un servicio de alojamiento de códigos nos ofrece muchas opciones necesarias para el trabajo colaborativo como:
• Permitir al grupo colaborativo detallar un problema, aportar sugerencias y realizar cambios en la plataforma simultáneamente.
• Realizar mantenimiento a las wiki en las distintas versiones de las páginas.
• Realizar revisión de los códigos, añadir una anotación a un fichero donde se desea y realizar cambios si se requiere mediante la realización de un commit específico.
• Funciona como un visor de ramas, donde se pueden evidenciar el progreso de proyectos creados.
• Sincronización de nuestra cuenta local con la nube de cambios realizados en cada proyecto de supositorios creados.

¿QUÉ ES HTML?

(cuarta lectura).

HTML es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

 

¿Qué es html5?

Es una la más reciente versión de HTML, que ofrece múltiples herramientas en la creación de páginas web, cuenta con nuevos elementos, atributos y comportamientos, en la edición de dichas páginas. Este conjunto de herramientas tecnológicas facilitan el desarrollo de un sitio web, entre las principales características que emplea encontramos:


Semántica: Permite describir de manera precisa el contenido presentara el sitio web.

Conectividad: Permite comunicarse fácilmente con el servidor.

Sin conexión y almacenamiento: Permite al sitio web almacenar sus datos localmente y operar sin conexión de manera más eficiente.

Multimedia: soporta multiplicidad de contenido multimedia como audio y video.

Gráficos y efectos 2D/3D: permite incorporar gráficos en la web como; lo son canvas 2D, WebGL, SVG, etc.

Rendimiento e Integración: garantiza una velocidad y manejo del hardware.

CSS3: permite crear distintas opciones para hacer diseños más innovadores.

CCS

(quinta lectura).

(Hojas de estilo en Cascada). CSS, es una tecnología que nos permite crear páginas web de una manera más interactiva. La multiplicidad de recurso es cada vez más extensa, con la aparición de nuevas versiones, que facilitan obtener resultados satisfactorios en el manejo de dichos recursos, que no se podían adquirir solamente con HTML, en la incorporación de márgenes, tipos de letra, fondos y colores.

¿QUÉ ES CCS3?

Conocidas comúnmente como hojas de estilo de cascada nivel tres, esta nueva versión propone múltiples reglas y estilos de contenidos que no permite una mayor diversidad de arreglos en una presentación de una página web. Este lenguaje permite configurar el estilo o la apariencia de las páginas web. Entre dichas modificaciones podemos encontrar el sombreado y redondeado en una página web.

¿Para qué sirve?

CSS3 sirve para definir la estética de una página web sitio web, sin tocar para nada los documentos HTML o JSP o ASP que lo compongan, y a eso se suma muchos más utilidades llamativas como sobras, transformaciones de figura, bordes y efectos 3D.
